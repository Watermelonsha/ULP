# University Life Planner

## Description

This is a simple desktop application written in Python using CustomTkinter.
The program helps students manage their university life: schedule, notes, and events.

## Features

* Add, view, and delete classes (schedule)
* Add, view, and delete notes
* Add, view, and delete events
* Data is saved in text files
* Simple and user-friendly interface

## Files

* schedule.txt – stores classes
* notes.txt – stores notes
* events.txt – stores events

## How to Run

1. Install required libraries:
   pip install customtkinter tkcalendar

2. Run the program:
   python main.py

3. Use buttons in the menu to navigate:

   * Schedule
   * Notes
   * Events

## Input Rules

* Time must be in format HH:MM
* Date is selected from calendar
* Only letters and numbers are allowed in text fields
